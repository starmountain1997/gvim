# g-claude

帮助用户配置 Claude Code 的项目，包含自动化安装脚本和配置模板。

## gen-commit-msg.md

`gen-commit-msg.md` 是一个用于生成 commit message 的 prompt，来源于 Reddit 讨论：[A useful prompt for commit message generation](https://www.reddit.com/r/ClaudeAI/comments/1l82cud/a_useful_prompt_for_commit_message_generation/)。

使用方式：在 Claude Code 中使用 `/gen-commit-msg` 命令时自动调用。
